HTTP详细介绍

超文本传输协议(HTTP),是一种用于客户端和服务端之间的应用层通信协议.其特点是简单快速、灵活.HTTP是一种无状态协议.HTTP协议不对请求和响应之间的通信状态进行保存,不保留一切请求和响应的信息
